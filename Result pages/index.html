Optimal Divide-and-Conquer to Compute Measure and Contour for a Set of Iso-Rectangles


The divide-and-conquer paradigm is often used to find an optimal solution of a problem. Its basic idea is to decompose a given problem into two or more similar, but simpler, subproblems, to solve them in turn, and to compose their solutions to solve the given problem. Here, we use the above paradigm to solve "The measure problem " and "The contour Problem" of a set of Iso-Rectangles(axis aligned rectangles). The algorithm followed was given by Ralf Hartmut Guting and has a time and space complexity of O(nlogn) where n is the cardinality of the set of rectangles.


Issues :

  • The order of the input of a particular rectangle formed by the lines x=a , y= b , x=c and y=d where a < c and b < d should specifically be "a b c d".

  • The algorithm assumes non degeneracy of rectangles meaning that the rectangles will not degerate into a line segment or a point.

  • The code does not account for decimal co-ordinates as vertices of the rectangles.


References :
